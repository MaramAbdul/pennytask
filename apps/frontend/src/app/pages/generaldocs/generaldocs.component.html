PennyTask - General Documentation

Welcome to the PennyTask documentation. This project is built using a **Monorepo Architecture** with **Nx**, where both the **Frontend (Angular)** and **Backend (NestJS + MongoDB)** are managed within the same repository.

🛠️ Technology Stack

  • 🔹 Monorepo: Nx Workspace
  • 🔹 Frontend: Angular (Standalone API)
  • 🔹 Backend: NestJS
  • 🔹 Database: MongoDB (Using Mongoose ORM)
  • 🔹 State Management: NgRx (Redux for Angular)
  • 🔹 Authentication: JWT-based authentication (8-hour sessions)
  • 🔹 Styling: Tailwind CSS

📂 Monorepo Structure

/pennytask
│── apps/
│   │── backend/        # NestJS Backend (MongoDB)
│   │── frontend/       # Angular Frontend
│── libs/               # Shared Libraries (if any)
│── .nx/                # Nx Configuration
│── package.json        # Dependencies
│── nx.json             # Nx Settings
│── tsconfig.base.json  # TypeScript Configuration

🛢️ Database Configuration (MongoDB)

We use **MongoDB** as our primary database with **Mongoose** as the ORM.

  • The database connection string is configured in the `.env` file.
  • All data models are defined in the `models/` directory.
  • MongoDB stores users and session data.

🔗 Backend & Frontend Communication

The **Frontend (Angular)** communicates with the **Backend (NestJS + MongoDB)** via REST API calls.

  • The **Frontend** makes HTTP requests using `HttpClientModule`.
  • The **Backend** provides APIs using `@nestjs/common`.
  • All requests are secured using **JWT authentication**.

⚙️ Development Setup

Follow these steps to set up the project locally.

🔹 Step 1: Install Dependencies

$ npm install

🔹 Step 2: Run Backend

$ npx nx serve backend

🔹 Step 3: Run Frontend

$ npx nx serve frontend

🔹 Step 4: Open in Browser

http://localhost:4200

🔐 Authentication Overview

We use JWT authentication with an 8-hour session expiration.

  • Users authenticate via **POST /api/auth/login**.
  • Tokens are stored in **LocalStorage** on the frontend.
  • Protected routes require **Authorization: Bearer [TOKEN]**.
  • Sessions expire after **8 hours**, requiring re-login.

📌 Notes

  • Ensure `.env` is properly configured for MongoDB connection.
  • Use **NgRx Selectors** to access user authentication state.
  • Run `npx nx serve backend` and `npx nx serve frontend` to start development.
